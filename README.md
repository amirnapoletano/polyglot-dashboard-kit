# Polyglot Dashboard Kit

A **framework-agnostic SaaS dashboard UI kit** built with clean HTML, CSS, and minimal JavaScript.

Designed for founders, indie hackers, and studios who want a **production-ready dashboard UI** without committing to a framework.

---

## Who this is for

- Indie hackers shipping MVPs fast
- Founders building internal tools or SaaS products
- Agencies reusing dashboard UI across multiple projects
- Developers who want **full control** over markup and styling
